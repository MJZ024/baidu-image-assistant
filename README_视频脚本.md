## 片段2 — 项目背景与技术介绍（约40秒）

```
首先说一下项目的技术背景。
随着人工智能技术的快速发展，计算机对图像的理解能力已经达到了很高的水平。
这次作业我选择接入百度AI开放平台的接口，原因有三个：

第二，OCR 文字识别，可以从图片中提取文字；
第三，语音合成 API，把文字转成自然语音进行朗读。
整个项目用 Python 实现，图形界面用的是 tkinter，这是 Python 内置的 GUI 库，不需要额外安装。
```

## 片段3 — 项目结构与代码演示（约40秒）

```
现在我们来看一下代码结构。
整个项目一共有 5 个文件：
config.py 是配置文件，存放百度 AI 的 API 密钥和语音参数；
baidu_api.py 是核心模块，封装了对百度各个 API 的调用；
main.py 是图形界面，用 tkinter 构建，包含图片上传、结果展示和语音播放等功能；
requirements.txt 是依赖列表；
README.md 是项目说明文档。

下面我们打开 PyCharm 看一下具体的代码实现。
首先是 baidu_api.py，这里我封装了多个 API 调用函数：
detect_objects 函数用于通用物体识别，
detect_animal 函数用于动物识别，
detect_text 函数用于 OCR 文字识别，
text_to_speech 函数用于语音合成，
full_analysis 是主函数，整合所有 API 的结果，一键完成从分析到解说的全部流程。
```

## 片段5 — 核心代码解析（约40秒）

```
接下来我们看一下核心代码的实现细节。

首先是API调用部分。
所有API调用都需要先获取 Access Token，这是百度的身份认证机制，
token有效期是30天，获取一次可以重复使用，避免频繁请求。

图像识别部分，我将图片转换为 Base64 编码后，
通过 POST 请求发送到百度的图像识别接口，
返回的结果包含识别到的物体名称和置信度分数。

语音合成部分，我设置了好几个参数：
per 参数控制音色，这里用的是女声；
spd 参数控制语速，pit 控制音调，vol 控制音量；
这些参数都放在了 config.py 里，可以灵活调整。

最后，系统根据分析结果，自动拼接成一段自然语言解说词，
再调用语音合成接口，生成 MP3 音频文件，由 winsound 模块在 Windows 上播放。
```
